# Kairos College Rides Organizer

A text messaging bot that automatically organizes rides for those who are off-campus.

## Getting Started

### Setup

Requires Python 3.10+ and a virtual environment.

```bash
pip install wheel
pip install -r requirements.txt
```

### Running

```bash
python app.py
```
